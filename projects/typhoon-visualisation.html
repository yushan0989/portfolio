Typhoon Forecast Visualisation

API Integration ¬∑ Geospatial Visualization ¬∑ Interactive Dashboard ¬∑ Workflow Optimization

Project Description & Objective

Developed an interactive geospatial dashboard to visualize typhoon trajectories, providing critical decision support for Emergency Operations Centers. The tool features an interface where users can manually adjust the path and wind radii, significantly increasing workflow efficiency for generating specialized graphs for client reports. By translating complex meteorological data into clear visual insights, the platform allows users to rapidly assess the potential impact of typhoons on specific geographic regions.

Methods & Data

This project utilises real-time meteorological datasets integrated from international organisations via APIs. The raw data undergoes a rigorous cleaning and standardisation process to ensure consistency across various text and numerical formats. The platform is built using R Shiny and leverages the Google Maps Geocoding API for high-precision geospatial mapping. Key technical features include the ability to edit typhoon parameters like latitude, longitude, and wind radii for dynamic path adjustments and detailed forecast analysis.
